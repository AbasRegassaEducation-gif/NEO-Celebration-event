▸ One-time setup needed

Connect a database first.

This app needs a free Firebase database to share submissions across devices. Takes about 5 minutes.

  1. Go to console.firebase.google.com and sign in with a Google account.
  2. Click Add project → name it (e.g. charter-detective) → skip Google Analytics → Create project.
  3. In the left sidebar, go to Build → Realtime Database → Create Database. Pick any location, then choose Start in test mode.
  4. Click the gear icon (top left) → Project settings. Scroll to Your apps and click the </> (web) icon.
  5. Give the app a nickname → Register app. Firebase will show you a firebaseConfig block.
  6. Open this HTML file, find the firebaseConfig block near the bottom of the <script>, and replace the placeholder values with yours.

Paste-ready format Firebase will give you:

const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "your-project.firebaseapp.com",
  databaseURL: "https://your-project-default-rtdb.firebaseio.com",
  projectId: "your-project",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "1234567890",
  appId: "1:1234..."
};

Once pasted, save the file, refresh the browser, and you're live.

Case № 2026/014 · Westview Discovery Academy

The school that started losing.

Eyes Only

Westview Discovery Academy is a K–8 charter school in its sixth year of operation. When it opened in 2020, it had a 200-family waitlist and a reputation for project-based learning. Today the picture looks different.

Enrollment has fallen for two consecutive years. Two of its most respected veteran teachers resigned in February citing burnout and pay. Standardized test scores dropped below the district average for the first time. Parents have started speaking up at board meetings, and the authorizer's renewal review is in nine months.

Connect a database first.

This app needs a free Firebase database to share submissions across devices.

  1. Go to console.firebase.google.com and sign in.
  2. Create a project, add a Realtime Database in test mode, and register a web app.
  3. Paste your firebaseConfig values into the script block at the bottom of this HTML.
